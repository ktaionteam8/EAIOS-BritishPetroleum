# infrastructure/kubernetes

Kubernetes manifests for staging and production deployments.

## Contents
- `namespaces/`    — K8s namespace definitions per domain
- `deployments/`   — Agent deployment manifests
- `services/`      — Service and ingress definitions
- `configmaps/`    — Environment configuration
- `secrets/`       — Secret references (Azure Key Vault)
- `hpa/`           — Horizontal pod autoscaler configs
