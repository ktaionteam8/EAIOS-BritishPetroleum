# 04 — Commercial & Trading

**Domain:** Commercial & Trading
**System:** EAIOS-BritishPetroleum

## Overview
AI agents for crude oil trading analytics, carbon credit markets, Castrol pricing, aviation fuel, LNG trading, and cross-commodity arbitrage.

## Agents
| Agent | Purpose |
|---|---|
| crude-trading-analytics-agent | Analyses crude oil market dynamics and trading opportunities |
| carbon-credit-trading-agent | Manages carbon credit portfolio and compliance trading |
| castrol-pricing-engine-agent | Optimises Castrol lubricant pricing globally |
| aviation-fuel-forecasting-agent | Forecasts aviation fuel demand and supply |
| lng-trading-platform-agent | Supports LNG cargo trading and optimisation |
| cross-commodity-arbitrage-agent | Identifies arbitrage opportunities across commodities |

## Structure
```
04-commercial-trading/
├── agents/         ← 6 AI agents (each with src/, tests/, config/)
├── applications/   ← Web/UI applications for this domain
├── services/       ← Shared microservices
└── models/         ← Shared data models and schemas
```
